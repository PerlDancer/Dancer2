=head1 Dancer2 Keyword Guide

This guide categorizes all Dancer2 keywords with descriptions, usage
guidance, and relationships with other keywords. Each entry links to the
relevant section in the L<Dancer2 Manual>.

=head2 Routing Keywords

=head3 L<get|Dancer2::Manual/get>

Defines a route that responds to HTTP C<GET> requests. Automatically
creates a C<HEAD> route.

B<When to use>: Fetch static resources or perform read-only actions.

B<Related Keywords>: C<post>, C<put>, C<del>, C<patch>, C<options>, C<any>.

=head3 post

Defines a route that responds to HTTP `POST` requests, typically for form submissions or resource creation.

=over

=item *

L<POST Route|https://metacpan.org/pod/Dancer2::Manual#post-route>

=back

B<When to use>: Submit forms, upload files, or create resources.

B<Related Keywords>: C<get>, C<put>, C<del>, C<patch>, C<options>, C<any>.

=head3 put

Defines a route that responds to HTTP `PUT` requests, often for updating resources.

=over

=item *

L<PUT Route|https://metacpan.org/pod/Dancer2::Manual#put-route>

=back

B<When to use>: Update existing resources in an idempotent way.

B<Related Keywords>: C<get>, C<post>, C<del>, C<patch>, C<options>, C<any>.

=head3 del

Defines a route that responds to HTTP `DELETE` requests.

=over

=item *

L<DELETE Route|https://metacpan.org/pod/Dancer2::Manual#delete-route>

=back

B<When to use>: Delete resources.

B<Related Keywords>: C<get>, C<post>, C<put>, C<patch>, C<options>, C<any>.

=head3 patch

Defines a route that responds to HTTP `PATCH` requests, often for partial updates.

=over

=item *

L<PATCH Route|https://metacpan.org/pod/Dancer2::Manual#patch-route>

=back

B<When to use>: Partially update existing resources.

B<Related Keywords>: C<get>, C<post>, C<put>, C<del>, C<options>, C<any>.

=head3 options

Defines a route that responds to HTTP `OPTIONS` requests.

=over

=item *

L<OPTIONS Route|https://metacpan.org/pod/Dancer2::Manual#options-route>

=back

B<When to use>: Check available HTTP methods for a resource.

B<Related Keywords>: All HTTP methods.

=head3 any

Defines a route that responds to multiple HTTP methods.

=over

=item *

L<ANY Route|https://metacpan.org/pod/Dancer2::Manual#any-route>

=back

B<When to use>: Handle routes that accept multiple methods (e.g., `GET` and `POST`).

B<Related Keywords>: All HTTP methods.

=head3 prefix

Defines a common path prefix for grouped routes.

=over

=item *

L<Prefix|https://metacpan.org/pod/Dancer2::Manual#prefix>

=back

B<When to use>: Organize routes logically under a shared prefix.

B<Related Keywords>: All HTTP methods.

=head2 URL and Route Helper Keywords

=over

=item *

L<uri_for|https://metacpan.org/pod/Dancer2::Manual#uri_for>

=back

B<When to use>: Generate URLs for links and redirects.

B<Related Keywords>: C<uri_for_route>.

=head3 uri_for_route

Constructs a URI for a named route.

=over

=item *

L<uri_for_route|https://metacpan.org/pod/Dancer2::Manual#uri_for_route>

=back

B<When to use>: Generate URLs based on route names and parameters.

B<Best Practice>: Use C<uri_for_route> with named routes to ensure consistency even if route paths change.

B<Related Keywords>: C<uri_for>.
